# Aerodisk

### Настройка backend
```
cd backend
```
```
pip install -r requirements.txt
```
```
python manage.py migrate
```
```
python manage.py createsuperuser
```
#### Запуск 
```
python manage.py runserver
```

### Настройка frontend
```
cd frontend
```
```
npm install
```
#### Запуск 
```
npm run serve
```
### fix lint
```
npm run -- --fix
```
